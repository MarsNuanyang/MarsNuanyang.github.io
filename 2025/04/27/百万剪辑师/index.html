# 02-【思维】如何构建视频框架.mp4

# 构建框架

主题、素材、情绪、时长

# 学习框架

为片子划分段落，标注（内容、时长、情绪、目的）

# 常用开头

  1. 铺垫情绪（镜头 + 台词 + 音乐）
  2. 交代背景
  3. 吸引眼球

# 常用结尾

  1. 首尾呼应
  2. 升华主题
  3. 做个总结

# 提升效率 文字替代镜头

在精剪前，可以先用文字替代镜头，标注想要的内容、音乐情绪或者大概的包装风格、甚至是创意形式。

# 03-【技巧】如何高效的找素材挑素材

# 台词怎么找

  1. 电影电视剧类型可以去网站上搜索
  2. 实拍或素材可以将使用剪映等支持音频识别的软件识别文字再导出

# 适合剪辑的镜头类型

  1. 高光时刻、名场面
  2. 有声音、有动作的镜头
  3. 人物镜头
  4. 相同景别

# 选镜头

  1. 边看边想
  2. 缺啥找啥
  3. “公式化” 看素材（高光时刻等）

# 快速找镜头

通过台词找

# 04-【选音乐】学会这几招，轻松找到完美 bgm

# 在哪找音乐

  1. 付费网站
  2. 免费网站

# 如何快速找到想要的音乐

  1. 情绪关键词、专辑等
  2. BPM 节拍数（BPM 越大，节奏越快）
  3. 通过音乐波形查看节奏

# 常用的音乐类型

预告片风格、歌词、翻唱

# 05－【剪音乐】如何把音乐变成你想要的样子 (时长、情绪、停止)

  1. 衔接点，对齐波形，做好过渡渐变
  2. 根据片子情况选择段落或叠加在一起使用
  3. 重新混合（无歌词 ）基本声音 - 音乐 - 持续时间 - 重新混编 - 输入时长

如何停止：

  1. 剪断素材，使用关键帧逐步递减，再加入室内混响等效果，数值：衰减、湿度拉高
  2. 使用其他声音（音效、台词等）

# 06-【剪音乐】不同音乐如何衔接

  1. 停止 + 转移注意力 + 开始（可以是暂停音乐 + 台词等）
  2. 切歌按钮
  3. 动作（带上耳机）
  4. 台词（听首歌吧 + 动作）
  5. 音乐变速改变情绪

# 07-[基础】 常见的镜头组接，让视频更丝滑

# 卡点剪辑

  1. 镜头和音乐鼓点对齐（可快不可慢）
  2. 动作卡点（产生声音 / 发生变化的一瞬间对齐鼓点）

# 匹配剪辑

形状、动作、台词、构图、色彩、景别、主体、角度等

# 动作匹配

动作可以是相同类似的（切菜、剁肉），也可以是按顺序的（起跑 -> 加速），需要注意运动方向

# 台词匹配

（星游记）

# 视线跟踪

在做混剪时，镜头切换焦点位置尽量相同（焦点会随着画面变动）

# 08【进阶】打破常规，但能玩出惊喜的组接

# 近乎万能卡点技巧

通过变换 节拍器 的音效速度对齐音乐节奏

# 变速剪辑

单个镜头将想表现的时间段加速减速

多个镜头之间可控制快慢，进行对比

# 跳切剪辑

删掉冗长 / 无聊的部分，保留重点部分

效果：压缩时间 / 制作节奏 / 反转对比 / 表现情绪

# 快剪快切

在一段比较长、缓的镜头后，加入多个短、快的镜头（特写、主体、声音、动作）
就可以产生对比反差，从而打破节奏（加入放大、旋转、裁切的一些效果），还要加音效

# 09-【蒙版】让视频迅速加分的 " 剪辑魔法”

# 抠像

AE Roto笔刷

# 创意分屏

左右各一半 ：对比（电影与现实的片段）、相似（动作、对话、构图）、关联（对话）

如何让画面完美结合

前期拍摄：构图相似

后期创作：裁切、旋转、缩放，或去原视频中找类似背景进行填充

# 10-【基础】常用的音效类型和使用技巧

音效类型：拟音（后期添加的）、效果音效（不太好描述由什么产生）、环境音

音效网站：envato、淘声网、爱给网

配音网站：Fliki - Turn text into videos with AI voices

# 11-【实战】如何恰当的使用音效

环境声 - 主体声 - 次要元素 - 效果音

有动作的下一个镜头可以添加放大等效果，例如：踩油门后，下一个镜头接仪表盘放大

# 12-【基础】什么是包装，我常用的包装有哪些

国风：泼墨、宣纸

抠图：超级键、颜色键、混合模式

# 13 - 实战】性价比很高的包装制作技巧

做包装思路：通过关键帧等让素材动起来、使用录屏获取想要的素材

跟踪贴图：

  1. 画选框进行跟踪
  2. 选择贴合范围
  3. 把跟踪数据应用到包装

# 14-【混剪作品】我的爆款是怎么做出来的

# 构思框架

开头 文案：【你记忆中的 2012 是什么样的】

Part1：

（快节奏、嗨，用来吸引注意力）

素材：动画、体育、电影、游戏、音乐等

Part2：

（舒缓、叙事）

素材：国内外大事件（新闻、玛雅语言、奥运会等）

Part3：

（渐进、高潮）

素材：积极的事件配合同期声总结 2012 这一年

结尾

素材：2012 年，人们对于未来的想象和期待

# 15-【商业视频】我是怎么做视频的

# 前期准备

制作周期：前期准备 - 初版 - 修改 - 定稿 - 上线

制作规格：帧率、尺寸、场景（线上发布 or 线下广告投放）

# 整理素材

  1. 客户提供
  2. 自己搜集（素材是否有授权、能使用，列清单）

# 构建框架

主题、情绪等，观众角度、甲方角度

# 剪辑阶段

沟通、粗剪不能太粗、找个观众

# 修改阶段

音乐版权等

# 输出成片

有无字幕、尺寸

# 16-【个人风格】如何养成个人风格

风格受什么影响？1. 所看片子类型 2. 所做的项目 3. 账号

如何养成个人风格：

  1. 看各种片子
  2. 放大、重复元素（节奏、音效、运镜、调色、文案、叙事方式、音乐类型）
  3. 自己想法

# 17-【最后三招】剪辑狮的最后三招

  • 做减法

    优化节奏 / 强化情绪 / 聚焦重点 / 创造留白

  • 加文案

    一字千金，提高片子含金量：引导、表达、总结、升华

  • 先完成，再完美

    提高效率 / 减少压力 / 减少决策 / 拥有更多的可能性

# 18-【直播答疑】视频导出后，我还会做什么

养成良好的习惯：导出视频时顺便输出单个轨道的内容（视频轨、音效轨、音乐轨、台词轨等）

# 21-【直播答疑】容易被音乐带着走怎么办

  1. 给音乐做剪辑（缩短、加长、调整段落）目的，为了让音乐服务于曲子
  2. 主动增加内容（台词、镜头、文案）
  3. 删减音乐内容（歌词、伴奏、旋律）
